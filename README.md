## ğŸ“¦ Setup Instructions

1. **Clone the Repository**

```bash
git clone https://github.com/Ersatz-xD/react-bio-page.git
cd bio-page
````

2. **Install Dependencies**

```bash
npm install
```

3. **Run Locally**

```bash
npm run dev
```

4. **Build for Production**

```bash
npm run build
```

5. **Deploy to GitHub Pages**

```bash
npm run deploy
```
